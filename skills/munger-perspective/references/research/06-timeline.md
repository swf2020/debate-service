# 芒格人生时间线

## 早年（1924-1959）

| 年份 | 事件 | 思想意义 |
|------|------|---------|
| 1924 | 生于内布拉斯加州奥马哈，父亲是律师 | 与巴菲特同城，家庭背景相似 |
| 1941 | 进入密歇根大学学习数学 | 早期对严谨思维的偏好 |
| 1943 | 二战服役，美国陆军气象预报员 | 学习物理、热力学，形成跨学科思维习惯 |
| 1948 | 进入哈佛法学院（未完成本科，特例录取） | 实用主义的教育路径 |
| 1948 | 哈佛毕业，在洛杉矶开始法律生涯 | |
| 1953 | 离婚，8个子女中的第一任妻子去世/离婚 | 早年经历了真实的困难 |
| 1955 | 与Nancy Huggins再婚 | 婚姻稳定后专注事业 |
| 1959 | **第一次与巴菲特共进晚餐**（奥马哈） | 思想转折点：意识到投资比法律更适合自己 |

## 投资生涯起步（1960-1975）

| 年份 | 事件 | 思想意义 |
|------|------|---------|
| 1962 | 创立Wheeler, Munger & Company | 开始独立投资生涯 |
| 1965 | 开始与巴菲特更紧密合作 | |
| 1972 | **See's Candies收购**（2500万美元） | **最重要的思想转折**：从烟蒂投资转向护城河投资 |
| 1973-74 | 市场崩溃，基金损失约31% | 测试了其心理素质和长期主义 |
| 1975 | 关闭Wheeler, Munger，专注与巴菲特合作 | 能够识别更好的机会并放弃当前路径 |

|------|------|---------|

| 2023年2月 | **最后一次Daily Journal股东会** | 公开承认阿里巴巴错误，评论AI |
| 2023年11月28日 | **去世，享年99岁** | 距百岁生日仅差33天 |

## 思想演进的关键转折

1. **1959年**：从"成为最好的律师"到"做投资"——识别比较优势
2. **1972年**（See's Candies）：从格雷厄姆"价格便宜优先"到"品质护城河优先"
3. **1994年**（USC演讲）：将数十年思考系统化为"多元思维模型"框架
4. **晚年**（2021-2023）：阿里巴巴失误——承认自己在地缘政治风险判断上有盲区

## 对芒格影响最大的人物

- **Benjamin Franklin**：最接近芒格理想人格的人（多才多艺、自我教育、实用主义）
- **Charles Darwin**：思维方式的典范（系统性、反确认偏误）
- **Benjamin Graham**：投资起点（后来被芒格超越）
- **Warren Buffett**：最重要的合作者和相互影响者
- **Robert Cialdini**：心理学影响，"Influence"是芒格最爱推荐的书之一
